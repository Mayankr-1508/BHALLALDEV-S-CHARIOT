# Bhallaldev's Chariot

**TA211: Manufacturing Processes I** | Prof. Shashank Shekhar, IIT Kanpur | Aug '25 – Dec '25

A functional war-chariot prototype inspired by cinematic design, integrating a gear-driven rotating chopper mechanism to demonstrate power transmission and motion conversion.

---

## Key Features

- **Chassis Construction** — Built using sheet metal forming, welding, and brazing on mild steel and GI sheets, constructing the base frame, side plates, seat, and structural supports
- **Precision Machining** — Components manufactured via drilling, turning, and plasma cutting, machining axles, wheel hubs, gears, and decorative profile features
- **Shaping & Finishing** — Moulding, casting, and forging techniques applied to shape wheels and joints, with grinding for dimensional accuracy on assembled parts
- **Motion Mechanism** — Linear knob motion converted into rotational chopper motion through a horizontal-vertical gear linkage

---

## Technical Specifications

### Chassis
- Base frame, side plates, seat, and structural supports fabricated from mild steel and GI sheets
- Joined using sheet metal forming, welding, and brazing

### Precision Components
- Axles, wheel hubs, gears, and decorative profile features manufactured via drilling, turning, and plasma cutting

### Forming & Finishing
- Wheels and joints shaped using moulding, casting, and forging
- Final dimensional accuracy achieved through grinding

### Mechanism
- Horizontal-vertical gear linkage converts linear knob motion into rotational chopper motion, integrating multiple manufacturing processes into one working mechanical system
